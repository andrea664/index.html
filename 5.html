ELEMENTOS QUE INTEGRAN EL DISEÑO Y DESARROLLO WEB
¿Qué es el desarrollo web? El proceso de crear sitios o aplicaciones web que involucra diseño, programación, contenido y configuración del servidor.
¿Qué es un Hosting? Servicio que permite almacenar y publicar un sitio web en internet.
¿Qué es un Dominio? Nombre único que identifica un sitio web. Ejemplo: .com
¿Qué es un Usuario? Persona registrada con nombre de usuario y contraseña para acceder al sitio.
¿Qué es una Contraseña? Clave secreta para proteger el acceso a una cuenta.
¿Qué es un Sitio Web? Conjunto de páginas relacionadas y accesibles desde una misma dirección.
¿Qué es una URL? Dirección web para acceder a una página o recurso. Ej: https://www.ejemplo.com
¿Qué es FTP? Protocolo para transferir archivos entre cliente y servidor.
¿Qué es HTML? Lenguaje de marcado que estructura el contenido en la web.

¿Qué es una etiqueta HTML?
Fragmento de código que indica al navegador cómo debe interpretarse el contenido. Las etiquetas van entre signos <>.

Tags Iniciales o de Raíz
<!DOCTYPE html> – Indica que se usa HTML5.
<html></html> – Raíz del documento.

Metadatos del Documento
<head></head> – Contiene metadatos del documento.
<title></title> – Define el título de la página.
<link> – Enlaza recursos externos (como CSS).
<meta> – Define información como codificación.
<style></style> – Incluye estilos CSS dentro del documento HTML.
